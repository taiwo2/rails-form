## Getting started

- Clone the repo `https://github.com/taiwo2/rails-form.git`
- `cd` into the directory
- Run `git checkout -b rails-form`
- Run `git pull origin rails-form`
- Run `bundle install`
- Setup your db, and run `rails db:create db:migrate`
